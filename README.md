# Nextcloud Custom Email Templates

Mit dieser Nextcloud-App kannst du als Admin über die GUI die Willkommens- und Passwort-Reset-E-Mail-Templates anpassen, mehrsprachig, mit CKEditor, Live-Vorschau und Backup.

## Features
- Admin-Only Einstellungen
- Bearbeiten von Willkommens- und Passwort-Reset-E-Mail-Vorlagen
- Mehrsprachigkeit
- Live-Vorschau inkl. Dummy-Platzhalter
- Sicherung und Wiederherstellung älterer Versionen (Backup)
- CKEditor (WYSIWYG)

## Installation
1. Dieses Verzeichnis als App in deinen Nextcloud `apps/`-Ordner kopieren.
2. Aktivieren im Nextcloud-App-Store oder per CLI.
3. Einstellungen findest du im Adminbereich.
